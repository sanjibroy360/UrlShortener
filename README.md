Follow these steps to set up the project locally:

1. **Clone the repository**

```bash

```

2. **Navigate to the Project Directory**

```bash
cd UrlShortener
```

3. **Install Dependencies**

```bash
yarn install
# or using npm:
# npm install

```

3. **Run Locally**

```bash
yarn dev
# or using npm:
# npm run dev

```
